# Code of Conduct

## Expected Behavior

- Be respectful and direct.
- Assume contributors are trying to improve the project.
- Keep disagreements focused on facts, tradeoffs, and project impact.
- Do not harass, threaten, or demean other contributors.

## Scope

This applies to repository issues, pull requests, discussions, and maintainer-controlled project spaces.

## Enforcement

Maintainers may edit or remove disruptive content, close issues, block users, or pause a discussion when needed to keep the project usable. Security reports should use the private reporting path in [SECURITY.md](SECURITY.md).
